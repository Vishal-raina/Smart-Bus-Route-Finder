# Smart-Bus-Route-Finder
A professional government-style frontend web application for searching Coimbatore district bus routes and timings with autocomplete functionality.


# Coimbatore District Bus Services 🚍

A professional government-style web application to search bus routes and timings in Coimbatore district.

## Features

- Search buses by From and To location
- Autocomplete suggestions
- Detailed timing information
- Clean government-style UI
- Responsive layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How to Run

1. Download or clone the repository
2. Open `index.html` in your browser

## Future Improvements

- Backend database integration
- Live bus tracking
- Mobile optimization
- API-based data loading
